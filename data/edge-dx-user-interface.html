        Note

        This article describes the user interface when you access Edge DX through Solve (solve.controlup.com) or direct tenant URL (<tenant-name>.sip.controlup.com). If you access Edge DX through ControlUp's DEX management platform (app.controlup.com), your interface will look slightly different, but the overall functionality of each page is the same.

        The Edge DX User Interface has four main sections: Dashboards, Devices, Reports and Configuration.

        If you sign in to Edge DX directly, rather than accessing Edge DX via Solve, you also see a My Account icon.

        Click the Dark / Light Mode toggle to switch between dark mode and light mode displays.

        Configuration

        The Configuration menu displays a complete or partial list of the Edge DX configuration options, depending on your permissions in Edge DX. For more information about permissions in Edge DX, see Role-Based Access in Edge DX.

        Click Configuration to open the Configuration menu:

        The following table provides additional details about the Configuration menu options:

        Edge DX Configuration OptionAdditional Details
        System EventsThe Configuration > System Events page displays and allows tracking of IT administration events such as administrator logons, device registration, errors, and more.
        AlertsOn the Configuration > Alerts page, set alerts to receive notifications about important events occurring across your physical device network. Refer to Edge DX Alerts for more details.
        ScriptsOn the Configuration > Scripts page, add new scripts and edit existing scripts. Refer to the Edge DX Scripting Guide for more details about scripts.
        DataThe Configuration > Data page displays the raw data collected from the distributed Edge DX Agents, which is processed and displayed in the Dashboards and in Reports. Refer to Custom Reports for more details on creating custom reports.
        DownloadsOn the Configuration > Downloads page, download the Edge DX Agent installer and view Agent installation instructions. Refer to Edge DX Agent Distribution and Installation for more details.
        Access ControlOn the Configuration > Access Control page, configure users and roles settings to manage and control user access to settings and features in Edge DX. Refer to Role-Based Access in Edge DX for more details.
        SettingsOn the Configuration > Settings page, configure various Edge DX settings in the Agent, Web Console, End User Activity, Network Latency, API Keys and Maintenance sections of the page.

        Dashboards

        When Edge DX opens, the Home dashboard displays.

        Click Dashboards to select the dashboard you want to view:

        For more information about the User Sentiment dashboards, see User Sentiment.

        The Edge DX Home dashboard displays widgets showing aggregate device data and metrics for the end user devices in your organization. The widgets are displayed in the Upper Display section of the Home dashboard, as shown in the screenshot below:

        The data displayed in the Managed Devices, Online Devices, and Active Applications widgets is the data collected during the indicated Date Range. The default Date Range is Today, but you can change it to one of several standard options (for example, Yesterday, Last 7 Days, This Month, etc.), or a non-standard period of time, as explained below.

        In the Lower Display section of the Home dashboard, an interactive map is displayed. Pins on the map indicate the locations of managed end user devices around the world. Click the pins to zoom in and see device data and performance metrics. Select a specific device detail or performance metric you want to view from the menu in the upper right-hand corner of the map.

        In addition to the Device Map display described above, it is also possible to view User Experience and Network data on the Home dashboard Lower Display. Click the Lower Display Option tab corresponding to the display you want to view.

        For more information about the data shown in the Network section, see Network Performance.

        Home Dashboard Upper Display

        The Upper Display section of the Home dashboard includes the following widgets showing aggregate device data and metrics for the end user devices in your organization:

        Widget LabelWidget DescriptionHoverClick
        Managed DevicesThe Managed Devices widget displays the total number of Managed Devices in your organization within the Date Range.Hover over the graphic display to see a breakdown of the Managed Devices by operating system.Click the Managed Devices graphic to open the Devices section of the Edge DX user interface.
        Online DevicesThe Online Devices widget displays a graph showing the number of Managed Devices in your organization which are online, at points in time along the timeline defined by the Date Range.Hover over the graphic display to see the details of the data points on the graph.Click the Online Devices graphic to open the Devices section of the Edge DX user interface.
        AlertsThe Alerts widget displays two values: the number of Alerts which occurred in the Last Hour and the number of Alerts which occurred in the Last 24 Hours.         Active ApplicationsThe Active Applications widget displays a bar graph showing user activity time on each of the five most used applications.Hover over the bars in the bar chart to see the details of the data behind the graphic display.Click any of the bars to open the End User Activity Report.

        Change the Date Range

        The default Date Range is Today, but you can change it to see data for a particular period of time that you select.

        To display Date Range data for a particular period of time, perform the following steps:

        1. Click the Date Range field. The Date Range selection dialog box opens:

        2. Select one of the Date Range selection options appearing in the Date Range selection dialog box. You can select one of the self-explanatory periods of time (i.e., Today, Yesterday, Last 7 Days, This Month, Last Month, or This Year). Another option is to select a Custom Range of days by selecting a start date and an end date for the Custom Range on the displayed calendar.

        3. When selecting a Custom Range only, you also have the option to select a customized start time and end time for the selected Date Range. If you want to change the daily start and end times from the default times of midnight to midnight (0:00 until 23:59), use the sliders at the bottom of the Date Range selection dialog box to set a custom start time and end time.

        4. Click Apply to apply your selections.

        Devices

        In the Devices section of the Edge DX User Interface, you can view various details about the managed devices in your organization, such as device identification and agent data. You can organize your view of the devices and search for particular devices by using the searching, sorting and filtering capabilities located below the column headings.

        Click the funnel icon below the Device, Location or Users column headings to open the filter builder and create advanced filters, if you do not want to use the &&, || and !! syntax in the header filters.

        Click the sorting icon below the Device Group column heading to select a sorting parameter.

        Use the horizontal scroll bar below the device data to scroll horizontally and view all columns.

        To show the columns containing Agent information, click the Show Agent Information checkbox .

        In columns containing key performance metrics, including Wi-Fi signal, network latency, CPU usage, memory available, logon times, and input delay, data is displayed using color-coded threshold indicators. Click these graphic displays to drill down and view additional device data.

        Device Drill Down

        In the Devices section of the Edge DX User Interface, click a device to drill down and view the individual device drill down page.

        On the individual device drill down page, click the section tabs to view the device data displayed in each section. Details about each section are provided in the table below.

        At the top of the device drill down page, click Actions, Assist, or Power to perform actions on the device. Click Settings to configure settings for the device.

        For more information on the actions available in the Assist menu, refer to Remote Assistance.

        Reports

        In the Reports section of the Edge DX user interface, you can access a variety of reports. Use Edge DX reports to perform extensive analysis of the data collected in Edge DX.

        Click Reports to Access the Reports section of the Edge DX user interface, where you can see the full list of reports available in Edge DX.

        Both preconfigured reports and customizable reports are available in Edge DX.

        For more details about customizable reports, see Custom Reports in Edge DX.
